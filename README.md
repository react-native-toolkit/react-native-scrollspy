# react-native-scrollspy

Implement scrollspy on React Native ScrollView

## Installation

```sh
npm install react-native-scrollspy
```

## Usage

```js
import Scrollspy from "react-native-scrollspy";

// ...

const result = await Scrollspy.multiply(3, 7);
```
